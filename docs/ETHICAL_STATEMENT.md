# Data Availability and Ethical Statement

## Data availability

The enterprise pilot data analyzed in the associated paper — including
practitioner interview transcripts, trust ratings, and interpretability scores
collected across the participating organizations — were gathered under
non-disclosure agreements. They contain confidential organizational information
and human-subject evaluations and cannot be publicly released. This is
consistent with standard practice in Design Science Research conducted in
real organizational settings.

The DBLP-ACM benchmark used for comparative evaluation is publicly available
through its original sources.

The synthetic dataset is produced by `data/synthetic/generate_synthetic.py`.
The published evaluation used a particular seed and parameter setting; users can
reproduce a comparable scale with `--entities 4500 --seed 42`. Because it is
fully generated by the provided script, the synthetic data is reproducible by
anyone.

## Reproducibility scope

The computational components of the framework — normalization, entity
resolution, knowledge graph construction, and the annotation control logic —
are fully reproducible from this repository on the synthetic demonstration data
and on the public benchmark. The quantitative figures in the paper depend on
the original study data and on human evaluations, and are therefore not
reproducible from this package alone. The repository reproduces the *method*,
including the grid-search procedure for parameter selection, rather than the
absolute reported scores.

## Responsible LLM use

Module 4 governs LLM annotation with three interception mechanisms
(fact-checking, confidence scoring, and human-review routing for sensitive or
low-confidence output). These are intended to reduce, not eliminate,
hallucination risk. Any deployment on real data should retain human oversight
for sensitive categories.
